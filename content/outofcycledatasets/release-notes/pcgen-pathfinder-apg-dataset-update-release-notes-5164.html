Pathfinder Advanced Players Guide
OOC Data Set for PCGen v5.16.4
Release Notes for Release 7

This is the sixth and final OOC update to PCGen's alpha data set for the Pathfinder Advanced Players Guide. It can be installed on top of all previous updates but doesn't require them.

Please note that this is a patch installer. In addition to the APG update it patches (overwrites) the Pathfinder Roleplaying Game Core Rulebook and Bestiary datasets with the latest data fixes we've made in our ongoing effort. Additionally, the latest Output Sheets, incorporating features supporting the Pathfinder data sets are included. The installer requires 5.16.4 and will not install in earlier versions.

This release is an alpha release. This "alpha" status refers to the status of the data set. As an alpha data set, this release is bound to contain errors. The PCGen team is dedicated to providing full support for the Pathfinder Roleplaying Game so we ask you, the PCGen community, to help us by putting the data set through its paces and reporting any bugs you find to the PCGen Data Team by posting your bug reports to PCGen@Yahoo! via email to our Yahoo! Group at pcgen@yahoogroups.com. Please include the tag "[PATHFINDER]" in the subject line so the data team can quickly identify your report.

NOTE: This is the last PCGen 5.16.4 compatible update to this collection of data sets. To take advantage of the continued improvement in the Pathfinder data sets you will need to upgrade to the latest Developmental release (PCGen 5.17.8+).

What are Out-of-Cycle Data Sets?

Out-of-Cycle (OOC) Data Sets are simply data sets which have been developed for the current production release after that version was released and are offered as separate downloads. Normally data sets are added to the program during the alpha development cycle, this is what would be called "in cycle".

A data set installation feature was added to PCGen in version 5.14. This feature makes it easy for users to download data set packages created for this purpose and install them into PCGen. The data sets in this release are compatible with PCGen 5.16.4 and later. Additional data sets may be added to this release as they are completed. If bugs are found or new features are added to a data set a new version of the data may be added to this release replacing the old version. The new version may be installed on top of the old version overwriting the previous files. Such changes will be noted in this document.

Installing Data Sets

To install any of these data sets follow these instructions.

  1. Download the data set packages you are interested in. The packages currently available can be found in the PCGen Stable Datasets folder. These files are zip files that have had the zip extension changed to pcz. DO NOT unzip these files. PCGen reads and unzips the archive files directly as part of the Data Install process.
  2. Launch PCGen and select Tools > Install Data.



  3. Click the Select button and navigate to the archive file to install via the presented "File Selection" dialog.
  4. PCGen will verify that the selected dataset is compatible with the version of PCGen being used.
  5. Information on the selected archive is displayed and you will be given a choice of "DATA" or "VENDORDATA" folders with the default specified in installer file.
  6. Select the destination and click Install.
  7. The dataset is saved to the selected location and is made available for use within PCGen.
