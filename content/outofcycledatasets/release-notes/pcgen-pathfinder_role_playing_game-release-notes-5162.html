Release Notes for Release 6

This is the sixth release of the data set for the Pathfinder Roleplaying Game Core Rulebook and the second to include the Pathfinder Bestiary. Also included are 4 conversion sets of Pathfinder Adventure Path dataset configured so they can be used in the Pathfinder gameMode. This installer contains the same dataset content as release 5 (just with numorous bug fixes) and does not require you to install release 5 first.

Please note that this is a patch installer, in addition to the new datasets it patches (overwrites) the Pathfinder Roleplaying Game Core Rulebook dataset as well as a number of other files so as to update 5.16.2 with the latest data fixes we've made in our ongoing effort. The installer requires 5.16.2 and will not install in earlier versions.

This sixth release is a beta release. This "beta" status refers to the status of the data set - this is NOT a data set for the "Pathfinder RPG Beta" rules. As a beta data set, this release is bound to contain errors. The PCGen team is dedicated to providing full support for the Pathfinder Roleplaying Game Core Rulebook so we ask you, the PCGen community to help us by putting the data set through its paces and reporting any bugs you find to the PCGen Data Team by posting your bug reports to PCGen@Yahoo! via email to our Yahoo! Group at pcgen@yahoogroups.com. Please include the tag "[PATHFINDER]" in the subject line so the data team can quickly identify your report.

Eddy Anthony - PCGen Pathfinder data monkey.

What are Out-of-Cycle Data Sets?

Out-of-Cycle (OOC) Data Sets are simply data sets which have been developed for the current production release after that version was released and are offered as separate downloads. Normally data sets are added to the program during the alpha development cycle, this is what would be called "in cycle".

A data set installation feature was added to PCGen in version 5.14.0. This feature makes it easy for users to download data set packages created for this purpose and install them into PCGen. The data sets in this release are compatible with PCGen 5.16.1 and above. Additional data sets may be added to this release as they are completed. If bugs are found or new features are added to a data set a new version of the data may be added to this release replacing the old version. The new version may be installed on top of the old version overwriting the previous files. Such changes will be noted in this document.

Installing Data Sets

To install any of these data sets follow these instructions.

  1. Download the data set packages you are interested in. Do not unzip them, PCGen reads the zip file directly so you need to retain that file.
  2. Launch PCGen and select Tools > Install Dataset.



  3. Click the Select button and navigate to the archive file to install via the presented "File Selection" dialog.
  4. PCGen will verify that the selected dataset is compatible with the version of PCGen being used.
  5. Information on the selected archive is displayed and you will be given a choice of "DATA" or "VENDORDATA" folders with the default specified in installer file.
  6. Select the destination and click Install.
  7. The dataset is saved to the selected location and is made available for use within PCGen.
